# Interactive PDF Mapper Installation Assets

## Asset Contents

### Sample PDF Files
- `sample-form.pdf` - Basic form with various field types
- `sample-section13.pdf` - Complex form with hierarchical fields (if available)
- `test-coordinates.pdf` - PDF for coordinate validation testing

### Configuration Templates
- `validation-template.json` - Validation rules and tolerance settings
- `performance-config.json` - Performance monitoring configuration
- `cache-strategy.json` - Cache optimization strategies

### Documentation
- `integration-guide.md` - Integration guide for existing React projects
- `api-reference.md` - API documentation for advanced usage
- `troubleshooting.md` - Common issues and solutions

### Environment Files
- `.env.template` - Environment variables template
- `docker-compose.yml` - Docker configuration for development
- `kubernetes/` - Kubernetes deployment templates

### IDE Configuration
- `vscode/` - VS Code extensions and settings
- `intellij/` - IntelliJ IDEA and WebStorm configuration
- `templates/` - Code templates and snippets

## Usage

These assets will be copied to the user's project during installation to provide:
1. Working examples and test files
2. Configuration templates for customization
3. Development environment setup
4. Integration examples and documentation

## Note

All assets are optional and will only be copied if the user requests them during installation.
